Basically this is a replacement for SpeakerBot with (mostly) full feature parity.  
It has a way to migrate data directly from SpeakerBot to make moving over (mostly) seamless.  
It supports lua extensions for adding new speech engines and such.  
It also has an in app way to add speech engines (though this may not support every engine ever.)  

Supports the following engines out of the box:
- SAPI5
- Azure
- Amazon Polly
- Google Cloud TTS
- Elevenlabs
- TTSMonster
- IBM Watson TTS
- Acapela Cloud
- CereProc Web Services
- Uberduck
- TikTok TTS (through a proxy, not sure if functional)
- FakeYou
- Fish Audio
- Inworld TTS
- Resemble AI
- Cartesia
- LMNT
